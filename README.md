# Para iniciar el proyecto

Tener el CLI de nest
```
npm -i -g @nestjs/cli
```
Levantar la base de datos
```
 docker compose up -d

```

Realiza la carga de la base de datos (Solo en desarrollo)
```
http://localhost:3000/api/v2/seed
```

* MongoDB
